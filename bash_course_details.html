🚀 Course Content - Basic To Advanced Bash Shell Scripting

Module 1: Introduction to Bash Shell Scripting

  • 🔹 What is Bash and Why Use Shell Scripting?
  • 🔹 Understanding Shells in Linux (Bash, Zsh, Ksh, etc.)
  • 🔹 Setting Up Your Environment (Ubuntu/CentOS)
  • 🔹 Running Bash Scripts (sh, bash, ./script.sh)
  • 🔹 Making Scripts Executable (chmod +x script.sh)
  • 🔹 Debugging and Troubleshooting Scripts (set -x, trap)

Module 2: Variables, Data Types, and Operators

  • 🔹 Understanding Variables ($VAR, export VAR)
  • 🔹 User Input Handling (read, read -p)
  • 🔹 Command Substitution ($(command), `command`)
  • 🔹 Arithmetic Operations (expr, (( )), bc)
  • 🔹 String Manipulation (${#var}, ${var:0:5})
  • 🔹 Working with Environment Variables ($PATH, $HOME)

Module 3: Control Flow - Conditional Statements & Loops

  • 🔹 if, elif, else Statements
  • 🔹 Case Statements (case, esac)
  • 🔹 Looping (for, while, until)
  • 🔹 Break and Continue Statements
  • 🔹 Using select for Menu-Driven Scripts
  • 🔹 Real-World Use Case: Automating User Account Creation

Module 4: Functions and Script Optimization

  • 🔹 Creating and Calling Functions
  • 🔹 Passing Arguments to Functions ($1, $2)
  • 🔹 Return Values and Exit Status (return, $?)
  • 🔹 Using Local and Global Variables (local var=value)
  • 🔹 Modular Scripting for Reusability
  • 🔹 Debugging Functions (set -x, trap ERR)

Module 5: Working with Files & Directories

  • 🔹 File Operations (touch, rm, mv, cp, ls)
  • 🔹 Directory Management (mkdir, rmdir, find, du)
  • 🔹 File Permission Handling (chmod, chown, umask)
  • 🔹 File Compression (tar, gzip, zip, unzip)
  • 🔹 File Content Manipulation (cat, tac, less, more)
  • 🔹 Automating Log Rotation and Cleanup Scripts

Module 6: Process Management & Job Automation

  • 🔹 Understanding Processes (ps, top, htop)
  • 🔹 Killing and Managing Processes (kill, pkill, killall)
  • 🔹 Background and Foreground Jobs (&, fg, bg, jobs)
  • 🔹 Scheduling Tasks with cron & at
  • 🔹 Automating Backups with Cron Jobs
  • 🔹 Monitoring Services with Bash Scripts

Module 7: Bash Scripting for Cloud Automation

  • 🔹 Introduction to AWS CLI
  • 🔹 Automating EC2, S3, and IAM with Bash
  • 🔹 AWS CLI + JQ + JMESPath for JSON Processing
  • 🔹 Parsing JSON with JQ & JMESPath
  • 🔹 Automating AWS IAM User Creation & Permission Setup
  • 🔹 Automating Cloud Storage & S3 Bucket Management

Module 8: Bash for DevOps & CI/CD Automation

  • 🔹 Introduction to DevOps with Bash
  • 🔹 Automating Git Operations (git add, commit, push)
  • 🔹 Docker Automation with Bash (docker build, docker push)
  • 🔹 Kubernetes (kubectl) Automation
  • 🔹 Using Bash Scripts in Jenkins Pipelines

Module 9: Executing REST APIs

  • 🔹 Working with REST APIs using curl & wget
  • 🔹 Parsing REST API Responses with jq
  • 🔹 Handling Authentication Tokens (Bearer, OAuth)
  • 🔹 Automating REST API Calls (e.g., GitHub API, AWS API)

Module 10: Advanced Bash Scripting Techniques

  • 🔹 Error Handling & Debugging (trap, set -e, set -o pipefail)
  • 🔹 Signal Handling (SIGINT, SIGTERM, SIGKILL)
  • 🔹 Parallel Processing (xargs -P, &, wait)
  • 🔹 Secure Scripting & Best Practices
  • 🔹 Logging & Monitoring with Bash Scripts
  • 🔹 Real-World Project: Automating a Server Health Check System
